# Admin Portal - E-commerce Management App

A React-based admin portal for managing products in an e-commerce application. It allows users to add, edit, view, and delete products with a responsive interface built using Tailwind CSS.

---

## Features

- Add new products
- Edit existing products
- Delete products
- Filter products by category
- Responsive design
- Clean UI using Tailwind CSS
- Routing with React Router
